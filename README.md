# אירוע הכפלה עסקית 2026 — דף נחיתה

דף נחיתה ממיר לאירוע "הכפלה עסקית 2026" (בית שמש · 20.07.2026).
בנוי ב-Next.js 16 · React 19 · Three.js (react-three-fiber) · Framer Motion.
עברית מלאה RTL, במיתוג הרשמי: שחור · זהב מטאלי · מוטיב XOXO, עם גיבור תלת-ממדי.

## הרצה מקומית

```bash
npm install
npm run dev      # פיתוח → http://localhost:3000
# או
npm run build && npm run start   # פרודקשן
```

## ⚙️ מה צריך לעדכן לפני עלייה לאוויר

הכול מרוכז בקובץ אחד: **`lib/site.ts`**

1. **`CHECKOUT_URL`** (שורה ראשונה) — הדבק כאן את **לינק רכישת הכרטיסים / התשלום**.
   כל כפתורי ה-CTA בדף, הפס הדביק, הפופאפ וכפתור "השלמת רכישה" שאחרי הטופס יובילו לשם.
   כרגע מוגדר `#register` (גלילה לטופס) עד שתדביק לינק אמיתי.
2. שאר הטקסטים, שמות המשתתפים, לוח הזמנים והשאלות הנפוצות — גם הם ב-`lib/site.ts`.

## מבנה
- `app/` — layout (RTL, פונט Assistant, מטא) + page (הרכבת הסקשנים)
- `components/` — Hero + Scene (תלת-ממד), sections, טופס, FAQ, פס דביק, פופאפ יציאה, ניווט
- `lib/site.ts` — כל התוכן והקונפיגורציה
