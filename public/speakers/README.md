# תמונות המרצים

כדי להוסיף תמונה של מרצה — פשוט שים כאן קובץ תמונה בשם המדויק שבטבלה.
**לא צריך לגעת בקוד.** אחרי שמעלים ל-GitHub (git push) התמונה תופיע אוטומטית.

## המלצות לתמונה
- פורמט: **JPG** (אם יש לך PNG — אפשר, אבל אז עדכן ב-`lib/site.ts` את הסיומת).
- יחס מומלץ: **פורטרט** (למשל 800×1000 פיקסל, יחס 4:5).
- רקע נקי, פנים ממורכזות למעלה.
- עד שמעלים תמונה — מוצג עיגול זהב ממותג עם ראשי התיבות של המרצה.

## מיקום להוספה מהירה
פשוט גררו את קובצי התמונות לתיקייה הזו (`public/speakers/`), עם השמות מהטבלה.
